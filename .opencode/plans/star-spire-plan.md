# 星际尖塔 (Star Spire) - 完整开发计划

## 项目概述
- **类型**: Roguelike 卡牌构筑游戏（类杀戮尖塔）
- **主题**: 科幻未来风 - 赛博朋克太空
- **技术**: 纯 HTML + CSS + JavaScript，浏览器直接运行
- **背景故事**: 人类最后的星际方舟"希望号"被外星AI控制，玩家是一名觉醒的赛博战士，需要穿越飞船3个区域（引擎层→核心区→舰桥），击败失控AI"奥米伽"
- **单局时长**: 15-30分钟

## 项目结构
```
game2/
├── index.html          # 主页面（包含所有游戏界面）
├── css/

├── js/

│   ├── enemies.js      # 敌人数据定义和AI逻辑
│   ├── relics.js       # 科技模块（遗物）系统
│   ├── events.js       # 随机事件系统
│   ├── shop.js         # 商店系统
│   ├── combat.js       # 战斗核心系统
│   ├── map.js          # 地图生成和导航
│   ├── ui.js           # UI渲染和动画

```

## 核心游戏机制

### 战斗系统
- **回合制**: 每回合抽5张牌，3点能量
- **卡牌类型**: 攻击牌（红）、技能牌（蓝）、能力牌（紫）
- **敌人意图显示**: 显示敌人下一步行动（攻击/防御/增益），玩家据此决策
- **护甲机制**: 每回合清零，需要先叠甲再挨打
- **状态效果**: 中毒/腐蚀（每回合掉血）、虚弱（伤害-25%）、力量（伤害+N）

### 地图系统
- 3层塔，每层约15个节点
- 节点类型：普通战斗、精英战斗、Boss、随机事件、商店、休息点、宝箱
- 分支路线选择（类似杀戮尖塔的树状地图）

## 视觉风格
- **背景**: 深蓝/深紫色 (#0a0a1a)
- **主色调**: 霓虹蓝 (#00d4ff) + 霓虹紫 (#b44aff)
- **卡牌**: 深色卡片 + 发光边框（攻击红/技能蓝/能力紫）
- **敌人**: emoji + CSS效果
- **字体**: 等宽字体，科幻感
- **动画**: CSS transition/animation（出牌滑动、受伤闪烁、护甲光效）

## 素材策略（两步走）
1. **第一步**: 先用 CSS + Emoji 快速开发，确保游戏完整可玩
2. **第二步**: 从免费素材站替换美化
   - Kenney.nl（CC0，完全免费）
   - OpenGameArt.org（CC0/CC-BY）
   - itch.io/game-assets/free

## 开发步骤

### Phase 1: 基础框架（核心可玩）— 9个文件
1. `index.html` — 所有游戏界面结构（标题/地图/战斗/奖励/事件/商店/休息/游戏结束/胜利 + 弹窗）
2. `css/style.css` — 科幻深色主题 + 霓虹光效 + 卡牌样式 + 响应式布局
3. `js/utils.js` — shuffle/randomInt/clamp/deepClone/delay 等工具函数
4. `js/cards.js` — 全部~50张卡牌数据定义 + 初始卡组配置 + 卡牌效果执行函数
5. `js/enemies.js` — 全部敌人数据 + AI行为模式 + 敌人行动执行
6. `js/combat.js` — 战斗核心（出牌/伤害计算/护甲/状态效果/回合管理/敌人行动/胜负判定）
7. `js/map.js` — 地图生成（节点+路线）+ 节点选择 + 层级推进
8. `js/ui.js` — UI渲染（卡牌显示/敌人显示/地图绘制/弹窗/动画效果）
9. `js/main.js` — 游戏状态管理 + 屏幕切换 + 流程控制（Game对象）

### Phase 2: 辅助系统 — 3个文件
10. `js/relics.js` — 科技模块数据定义 + 触发时机和效果
11. `js/events.js` — 随机事件数据 + 选择逻辑和效果
12. `js/shop.js` — 商店商品生成 + 买卖逻辑

### Phase 3: 打磨
13. CSS动画完善（出牌滑动、受伤闪烁、combo特效、卡牌hover放大）
14. 平衡性调整（数值微调）
15. 素材替换（如用户后续提供或下载免费素材）

## 验证方式
- 用浏览器直接打开 index.html
- 验证完整流程：标题 → 地图 → 战斗 → 奖励选卡 → 继续探索 → 事件/商店/休息 → Boss战 → 胜利/失败
- 测试边界情况：卡组为空、能量为0、HP为0、手牌满

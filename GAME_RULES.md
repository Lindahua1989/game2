## 游戏概述

### 背景故事
人类最后的星际方舟"希望号"被外星AI控制。你是一名觉醒的赛博战士，需要穿越飞船的3个区域（引擎层 → 核心区 → 舰桥），击败失控的主控AI"奥米伽"，夺回方舟的控制权。

### 游戏目标
- 穿越3层地图，每层15个节点
- 击败每层末尾的Boss
- 最终击败主控AI·奥米伽

### 初始状态
- **生命值**: 85/85
- **能量**: 3点/回合
- **初始金币**: 50
- **初始卡组**: 10张卡牌
  - 激光射击 ×5
  - 能量护盾 ×4
  - 等离子炮 ×1

## 基础规则

### 回合制战斗
每场战斗采用回合制：
1. **玩家回合**
   - 抽5张牌
   - 获得3点能量（可通过遗物/能力增加）
   - 打出卡牌消耗能量
   - 护甲在回合开始时清零
   - 点击"结束回合"按钮结束

2. **敌人回合**
   - 敌人按照显示的意图行动
   - 攻击、防御、施加状态等

### 能量系统
- 每回合开始获得3点能量
- 打出卡牌消耗对应能量
- 未使用的能量不保留到下一回合（除非有特殊能力）
- 能量可以通过卡牌、遗物等方式获得额外点数

### 护甲机制
- 护甲可以抵消受到的伤害
- **重要**: 护甲在每回合开始时清零
- 需要在敌人攻击前获得护甲
- 伤害优先扣除护甲，剩余伤害扣除生命值

### 抽牌与弃牌
- 每回合开始抽5张牌
- 回合结束时，手牌全部进入弃牌堆
- 抽牌堆为空时，弃牌堆洗牌后成为新的抽牌堆
- 可以在战斗界面查看抽牌堆和弃牌堆

## 战斗系统

### 卡牌类型

#### 攻击牌 (红色边框)
- 对敌人造成伤害
- 可以附加状态效果（腐蚀、虚弱等）
- 部分攻击牌可以攻击多个敌人

#### 技能牌 (蓝色边框)
- 获得护甲
- 抽牌
- 获得能量
- 回复生命
- 施加状态效果

#### 能力牌 (紫色边框)
- 提供持续性增益效果
- 一旦打出，效果持续整场战斗
- 例如：每回合获得护甲、增加伤害等

### 目标选择
- **单体目标**: 需要点击选择一个敌人
- **全体目标**: 自动对所有敌人生效
- **随机目标**: 随机选择敌人（多次攻击可能选择不同敌人）
- **自身**: 对玩家自己生效

### 伤害计算
```
实际伤害 = 基础伤害 + 力量加成 + 其他加成
最终伤害 = 实际伤害 - 敌人护甲（如果有）
```

### 多段攻击
- 某些卡牌显示"造成X伤害 xN"
- 表示攻击N次，每次造成X点伤害
- 每次攻击都会触发相关效果（如力量加成、荆棘反弹等）

## 游戏技巧

### 战斗技巧
1. **优先获得护甲**: 在敌人攻击前使用防御牌
2. **注意敌人意图**: 观察敌人下一步行动，做出最优决策
3. **合理使用能量**: 不要浪费能量，尽量每回合用完
4. **多段攻击很强**: 力量加成对多段攻击效果显著
5. **腐蚀是持续伤害**: 对高HP敌人使用腐蚀效果很好
6. **虚弱降低伤害**: 对高攻击敌人使用虚弱可以减少受到的伤害

### 构筑技巧
1. **保持卡组精简**: 不要无脑拿卡，保持卡组质量
2. **平衡攻防**: 攻击牌和防御牌要有合理比例
3. **能力牌很强**: 能力牌提供持续效果，优先获取
4. **移除基础牌**: 通过商店移除弱小的基础牌
5. **升级关键牌**: 优先升级核心卡牌

### 地图技巧
1. **规划路线**: 提前规划好要走的路线
2. **精英奖励好**: 精英战斗奖励更丰厚，但难度更高
3. **休息很重要**: 合理使用休息点回复或升级
4. **商店必去**: 商店可以移除卡牌，非常重要
5. **事件随机**: 事件有随机性，根据当前状态选择

## 常见问题

**Q: 护甲什么时候清零？**
A: 每回合开始时，玩家的护甲会清零。所以要在敌人攻击前获得护甲。

**Q: 腐蚀伤害怎么计算？**
A: 每回合开始时，受到等同于腐蚀层数的伤害，然后腐蚀层数减少1。

**Q: 多段攻击怎么计算力量加成？**
A: 每次攻击都会加上力量值。例如：力量+3，攻击4×2，实际造成(4+3)×2=14伤害。

**Q: 能量可以保留到下一回合吗？**
A: 默认不可以，除非有"超频引擎"遗物。

**Q: 怎么移除卡牌？**
A: 在商店花费75金币可以移除一张卡牌。

**Q: 升级卡牌有什么好处？**
A: 升级后的卡牌数值提升约30%，名称后面会显示"+"。

**Q: Boss有几个阶段？**
A: Boss有2-3个阶段，HP降低到一定程度会切换阶段，攻击模式会改变。

**Q: 怎么获得新皮肤？**
A: 通过达成特定成就解锁新皮肤。
